# Safe setup checklist

1. Create the Admin Safe and Treasury Safe on Sepolia using the Safe UI.
2. Set owners and thresholds for each Safe.
3. Record the Safe addresses in the local untracked `wallets.json` file.
4. Confirm the owners can sign and the threshold is appropriate for the planned operations.
5. Keep the Safe addresses and signer metadata out of version control.
6. Perform any later role grants manually and only on testnet.
